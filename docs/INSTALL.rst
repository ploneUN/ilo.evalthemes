ilo.evalvocabulary Installation
-------------------------------

To install ilo.evalvocabulary using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.evalvocabulary`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.evalvocabulary

* Re-run buildout, e.g. with:

    $ ./bin/buildout
